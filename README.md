# Abhijeet Gym Modern Website

Modern React + Vite promotional landing page for Abhijeet Gym And Fitness.

## Stack

- React
- TypeScript
- Vite
- Framer Motion
- React Three Fiber
- React Helmet Async
- CSS architecture with feature-based folders
- Vercel-ready deployment

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel login
vercel
vercel --prod
```

## Replace placeholder gallery

Add optimized gym photos inside `public/images/` and update `src/features/home/data/gallery.ts` plus `GallerySection.tsx` to render real images.
